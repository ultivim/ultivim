# ultivim

A Neovim distro for everyone.

## Features

- ~35ms startup time
- 85%+ lazy loaded plugins
- LSP, completion, formatting out of the box (format on save)
- Treesitter syntax highlighting
- Fuzzy finding with fzf-lua
- File navigation with oil.nvim
- Git integration with gitsigns + neogit
- Code outline with outline.nvim
- Rose Pine colorscheme

## Installation

You should clone our [starter](https://github.com/ultivim/starter.git) repo.

```bash
git clone https://github.com/ultivim/starter.git ~/.config/nvim
nvim
```
